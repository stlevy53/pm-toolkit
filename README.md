# PM Toolkit

A collection of tools built to make product managers more effective at the work that actually matters — communicating clearly, making defensible decisions, understanding customers, and staying sharp on the competitive landscape.

Each project in this portfolio addresses a real, recurring pain point in PM workflows. They're built with AI where it genuinely helps and kept simple where it doesn't.

## Projects

| Project | Problem | Status |
|---|---|---|
| [Audience Translator](briefs/audience-translator.md) | PMs rewrite the same content for different audiences manually, losing hours and introducing drift | Backlog |
| [Competitive Intel Tracker](briefs/competitive-intel-tracker.md) | Competitive docs go stale immediately and the research process restarts from scratch every quarter | Backlog |
| [PRD Quality Scorer](briefs/prd-quality-scorer.md) | PRD templates check for section presence, not content quality — bad PRDs pass every checklist | Backlog |
| [Decision Log](briefs/decision-log.md) | PM decisions go unrecorded and rationale disappears into Slack threads nobody can find | Backlog |
| [Customer Story Extractor](briefs/customer-story-extractor.md) | Customer insights are trapped in unstructured artifacts and rarely get synthesized into usable stories | Backlog |
